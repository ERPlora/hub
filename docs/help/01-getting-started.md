# Primeros Pasos

Bienvenido a **ERPlora**, tu sistema de punto de venta modular. Esta guía te ayudará a configurar y empezar a usar tu Hub.

## Requisitos del Sistema

| Requisito | Mínimo |
|-----------|--------|
| Navegador | Chrome 90+, Firefox 90+, Safari 15+, Edge 90+ |
| Conexión | Internet (el Hub requiere conexión al Cloud) |
| Resolución | 1024x768 (recomendado 1920x1080) |
| Sistema Operativo | macOS 12+, Windows 10+, Linux (Ubuntu 20.04+), Android 10+ |

## Acceder al Hub

1. Abre tu navegador y ve a la URL de tu Hub (ej: `https://mi-negocio.a.erplora.com`)
2. Introduce tu **PIN** de usuario
3. Ya estás dentro del dashboard principal

## Primer Uso: Asistente de Configuración

La primera vez que accedas, el asistente te guiará en 4 pasos:

1. **Región** — Selecciona tu país e idioma
2. **Tipo de Negocio** — Elige el sector y tipo (restaurante, tienda, clínica, etc.)
3. **Negocio** — Configura nombre, dirección, logotipo y datos fiscales
4. **Impuestos** — Configura clases de impuestos (IVA, tipos reducidos, etc.)

> El tipo de negocio determina qué módulos se instalan automáticamente y qué productos de ejemplo se cargan.

## Navegación

- **Barra lateral izquierda** — Acceso a Dashboard, Sistema, Empleados, Roles, Ajustes, Módulos y Marketplace
- **Módulos** — Aparecen en el menú superior una vez instalados
- **Barra de pestañas** — Cada módulo tiene sus secciones accesibles desde la barra inferior
