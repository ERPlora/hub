# Preguntas Frecuentes

## General

### ¿Necesito internet para usar ERPlora?
Sí, el Hub requiere conexión a internet para funcionar ya que se comunica con el Cloud de ERPlora.

### ¿Puedo usar ERPlora en el móvil?
Sí. El Hub es una aplicación web responsive que funciona en cualquier navegador moderno. Puedes instalarla como PWA para tener acceso rápido desde la pantalla de inicio.

### ¿Cuántos usuarios puedo tener?
Depende de tu plan. Los usuarios son locales al Hub, no cuentan como licencias separadas.

### ¿Puedo usar ERPlora en varios dispositivos a la vez?
Sí. Puedes tener un terminal principal con Bridge (impresora, cajón) y varios terminales satélite (tablets, móviles) accediendo al mismo Hub simultáneamente.

## Hardware

### ¿Qué impresoras son compatibles?
Cualquier impresora de tickets ESC/POS, que incluye la mayoría de marcas: Epson, Star, Bixolon, etc. Conexión por USB, red (Ethernet/WiFi) o Bluetooth.

### ¿Necesito el Bridge en todos los dispositivos?
No. Solo en el terminal principal donde esté conectada la impresora y el cajón. Los demás dispositivos acceden como terminales satélite.

### ¿Funciona el escáner de códigos de barras sin Bridge?
Sí. Los escáneres USB actúan como teclado y funcionan directamente en el navegador sin necesidad del Bridge.

## Datos

### ¿Dónde se guardan mis datos?
En una base de datos PostgreSQL en los servidores seguros de ERPlora en Europa (Alemania), con backups automáticos.

### ¿Puedo exportar mis datos?
Sí. El módulo de Exportación de Datos permite exportar a CSV, Excel y PDF.

### ¿Puedo hacer backups manuales?
Sí. Ve a Ajustes → Copias de Seguridad → "Ejecutar Backup Ahora".

## Soporte

### ¿Cómo reporto un problema?
Desde el menú lateral, pulsa **"Reportar un Problema"**. Se abrirá un formulario donde puedes describir el error.

### ¿Dónde encuentro actualizaciones?
Las actualizaciones se aplican automáticamente. Tu Hub siempre está en la última versión.
