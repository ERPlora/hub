# Ajustes

Desde **Ajustes** puedes configurar todos los aspectos de tu Hub.

## Datos del Negocio

Configura la información de tu empresa:
- **Nombre comercial** — El nombre que aparece en tickets y facturas
- **Dirección** — Dirección fiscal
- **NIF/CIF** — Número de identificación fiscal
- **Teléfono y email** — Datos de contacto
- **Web** — Sitio web (opcional)
- **Logotipo** — Se muestra en tickets, facturas y la app

## Configuración del Hub

- **Idioma del sistema** — Idioma por defecto del Hub
- **Zona horaria** — Para fechas y horas correctas
- **Moneda** — Moneda principal de operación
- **Tema** — Claro, oscuro o automático (según el sistema)

## Preferencias de Usuario

Cada usuario puede personalizar:
- **Idioma** — Idioma de la interfaz (independiente del sistema)
- **Avatar** — Imagen de perfil

## Impuestos

Gestiona las clases de impuestos:
- Crea diferentes tipos (IVA general 21%, IVA reducido 10%, IVA superreducido 4%)
- Marca uno como **predeterminado**
- Los impuestos se aplican automáticamente a los productos

## Copias de Seguridad

Configura backups automáticos:
- **Activar/desactivar** backups automáticos
- **Frecuencia** — Diaria, semanal o mensual
- **Hora** — A qué hora se ejecuta
- **Retención** — Cuántos días se conservan
- **Máximo** — Número máximo de copias
- **Backup manual** — Ejecutar una copia en cualquier momento

## Hardware (Bridge)

Configura la conexión con el hardware:
- **Activar Bridge** — Interruptor principal
- **Puerto** — Puerto WebSocket (por defecto 12321)
- **Auto-impresión** — Imprimir automáticamente al cobrar
- **Impresión satélite** — Permitir que terminales satélite impriman vía el Bridge principal
- **Impresoras** — Ver, configurar y gestionar impresoras conectadas
